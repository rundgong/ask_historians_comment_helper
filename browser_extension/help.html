Ask Historians Comment Helper

Browser extension to help displaying comment counts for questions at Reddit's AskHistorians subreddit.

This browser extension will show the number of top level comments next to the regular comment count.
In addition it will allow you to monitor up to 10 questions. The status of these questions will be shown at the icon in the toolbar.

On Mobile Reddit the comment helper will get triggered by scrolling the page.
On Old Reddit and New Reddit the comment helper will trigger on page load.
